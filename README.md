Shortcuts
---------
1. 90%

ALT + F4 : 활성창 종료

2. 70%
WIN + d : 바탕화면 바로보기
WIN + e : 내컴퓨터 열기

3. 50%
CON + tab : 브라우저 내부 탭 이동 
CON + w : 현재 실행중 탭 하나만 종료 

4. 10%
WIN + 방향키 : 창 분할 
CON + t : 브러우저 새로운 탭 띄우기 
CON + SHIFT + tab : 브라우저 닫힌 탭 띄우기
